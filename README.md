 Project Overview: Egypt House Price Prediction

 1. Dataset Description
The dataset contains information about various real estate properties in Egypt. It includes features such as:
Type: (Apartment, Villa, Duplex, etc.)
Bedrooms & Bathrooms: Count of rooms.
Area: Total area in square meters.
Location: City or neighborhood.
Target Variable:`Price` (Numerical value).
 2. Machine Learning Problem
  Type: Supervised Learning (Regression).
Goal: The objective is to predict the market price of a house based on its features. Since the output is a continuous numerical value, we are treating this as a Regression problem.
